# XAI-ADPKD-Net

This software package (XAI-ADPKD-Net) includes the source codes for a fully automated classification of typical (class 1) and atypical (class 2) morphology in subjects with autosomal dominant polycystic kidney disease (ADPKD).
Please note that, currently, the XAI-ADPKD-Net is under consideration for journal publications.
